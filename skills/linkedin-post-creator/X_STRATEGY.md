## 📝 Content Types for X

### Type 1: Thread (5-10 tweets)
**Best for**: Educational, process, detailed insights

```
Structure:
Tweet 1: Hook
Tweet 2-8: Body (1 point per tweet)
Tweet 9: Key takeaway
Tweet 10: CTA + LinkedIn link
```

### Type 2: Single Tweet (280 chars)
**Best for**: Quick tips, hot takes, engagement

```
Structure:
[Strong hook + emoji]
[1-2 sentences]
[Image required]
```

### Type 3: Quote Tweet
**Best for**: Adding opinion to others' content

```
Structure:
[Original tweet]
[Your take: Agree/disagree + why]
```

### Type 4: Reply
**Best for**: Engagement, building relationships

```
Structure:
[Answer the question]
[Add value]
[Ask follow-up]
```

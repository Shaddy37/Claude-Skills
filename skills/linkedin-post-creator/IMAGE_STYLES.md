## 🖼️ Image Style Templates

### Style 1: Case Study Dashboard

**Description**: Professional analytics dashboard showing metrics and results

**Visual Elements**:
- Dark theme background
- Green/red indicators for improvement
- Large bold metrics
- Before → After arrows
- Branded header

**Best Metrics to Show**:
- $ Saved per month
- % Efficiency improvement
- Time saved per week
- Number of workflows

**Dimensions**:
- LinkedIn: 1080x1350 (4:5)
- X: 1080x1080 (1:1)

---

### Style 2: Tool Stack Grid

**Description**: Clean grid of logos with tool names

**Visual Elements**:
- White/clean background
- 3x2 or 2x3 grid
- Tool logos centered
- Category labels
- Modern tech aesthetic

**Tools to Include**:
- n8n, Claude Code, Vapi
- MiniMax, Claude, Gemini
- Make, Zapier, viaSocket

**Dimensions**:
- LinkedIn: 1080x1350 (4:5)
- X: 1080x1080 (1:1)

---

### Style 3: Process Flow Chart

**Description**: Step-by-step visual with arrows

**Visual Elements**:
- Horizontal flow
- Numbered steps (1-5)
- Icons for each step
- Arrow connectors
- Blue gradient theme

**Steps Structure**:
- Analyze → Design → Build → Deploy → Optimize

**Dimensions**:
- LinkedIn: 1080x1350 (4:5)
- X: 1080x675 (2:1)

---

### Style 4: Myth vs Reality

**Description**: Split comparison cards

**Visual Elements**:
- Two-column layout
- Left: Myth (red)
- Right: Reality (green)
- Bold typography
- Contrast colors

**Best For**:
- Myth buster posts
- Comparison content
- Warning posts

**Dimensions**:
- LinkedIn: 1080x1350 (4:5)
- X: 1080x1080 (1:1)

---

### Style 5: Big Number Result

**Description**: Single metric prominently displayed

**Visual Elements**:
- Large bold number
- Metric description below
- Brand colors
- Minimalist design
- Impact statement

**Best Metrics**:
- 200+ Workflows
- $2,000+/Month Saved
- 85% Reduction
- 500+ Calls Handled

**Dimensions**:
- LinkedIn: 1080x1080 (1:1)
- X: 1080x1080 (1:1)

## 🎯 Color Schemes

### Scheme 1: Tech Blue (Recommended)
```
Primary: #0066CC (Blue)
Secondary: #00A3E0 (Cyan)
Background: #1A1A2E (Dark)
Text: #FFFFFF (White)
Accent: #00D4FF (Bright Cyan)
```

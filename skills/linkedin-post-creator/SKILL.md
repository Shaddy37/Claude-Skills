# 🤖 LinkedIn + X AI Automation Authority Builder

## What This Skill Does

Creates comprehensive content systems for AI Automation Experts that:
1. **Builds authority** through high-engagement LinkedIn posts
2. **Expands reach** with X (Twitter) integration
3. **Generates leads** through strategic CTAs
4. **Tracks competitors** for continuous optimization
5. **Monetizes engagement** toward $5K-$10K/month goal

## The Complete Workflow

### Phase 1: Input & Research
1. Accept topic, post type, or content goal
2. Load user profile data from PROFILE_ANALYSIS.md
3. Check competitor hooks from HOOKS_COLLECTION.md
4. Research trending topics if needed

### Phase 2: Content Creation
5. Select optimal post format (carousel, text, case study, etc.)
6. Choose hook type based on post goal
7. Create LinkedIn post (1300-2000 characters)
8. Adapt content for X (thread + single tweet)

### Phase 3: Image Generation
9. Select image style from IMAGE_STYLES.md
10. Generate LinkedIn image (1080x1350)
11. Generate X image (1080x1080)
12. Use Puter.js (free Nano Banana) for generation

### Phase 4: Output & Organization
13. Save all files to organized folder
14. Add to CONTENT_CALENDAR.md
15. Include monetization CTAs
16. Export for ClickUp if requested

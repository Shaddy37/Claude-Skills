## 🔄 The v4.3 Workflow (LinkedIn + X - 2x/day EACH)

### ⚠️ CRITICAL: ALWAYS CREATE BOTH
When user asks for a post, you MUST create:
1. **LinkedIn post** (with image prompt)
2. **X/Twitter post** (with image prompt)

Do not ask - just do both. This is mandatory.

### Phase 1: Strategic Inquiry (Mandatory)
Ask the user:
1. **Topic/Metric**: What specific result or topic are we featuring?
2. **Post Goal**: Authority (Case Study), Reach (Carousel), or Engagement (Opinion)?
3. **Lead Magnet**: Should we attach a specific free resource?
4. **Visual Choice**: Infographic (Canva-style) or Personal Photo with overlay?

### Phase 2: Library Synthesis
1.  **Hooks**: Pull from [HOOKS_COLLECTION.md](file:///e:/Antigravity/Shadab/.claude/skills/linkedin-post-creator/HOOKS_COLLECTION.md).
2.  **Formulas**: Pull from [POST_FORMULAS.md](file:///e:/Antigravity/Shadab/.claude/skills/linkedin-post-creator/POST_FORMULAS.md).
3.  **Posting Strategy**: Use [POSTING_STRATEGY.md](file:///e:/Antigravity/Shadab/.claude/skills/linkedin-post-creator/POSTING_STRATEGY.md).
4.  **X Strategy**: Use [X_STRATEGY.md](file:///e:/Antigravity/Shadab/.claude/skills/linkedin-post-creator/X_STRATEGY.md) for X adaptation.

### Phase 3: Build BOTH Posts
For EACH post (LinkedIn AND X):
- Write the post content
- Generate image prompt from [IMAGE_STYLES.md](file:///e:/Antigravity/Shadab/.claude/skills/linkedin-post-creator/IMAGE_STYLES.md)
- Select CTA from [CTA_LIBRARY.md](file:///e:/Antigravity/Shadab/.claude/skills/linkedin-post-creator/CTA_LIBRARY.md)

**Output format:**
```
## 📝 LINKEDIN POST
[Post content + image prompt]

## 🐦 X POST  
[Thread/single post + image prompt]
```

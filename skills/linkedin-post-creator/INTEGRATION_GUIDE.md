## 🎯 Overview

This guide explains how to integrate the content calendar with ClickUp for scheduling and managing posts.

**Note**: This is a future step. For now, use the CSV export and manual copy-paste.

---

## 📋 When Ready

### Step 1: Set Up ClickUp

1. Create a ClickUp account (if not done)
2. Create a new List: "LinkedIn + X Content"
3. Add custom fields:
   - Platform (dropdown: LinkedIn, X)
   - Post Type (dropdown)
   - Topic (text)
   - Hook (text)
   - CTA (text)
   - Status (dropdown: Draft, Scheduled, Posted)

### Step 2: Export Calendar

1. Go to export_template.csv
2. Review upcoming posts
3. Copy content for each post

### Step 3: Create Tasks

For each post:
1. Create task in ClickUp
2. Add due date/time
3. Add content in description
4. Attach images
5. Set status to "Scheduled"

### Step 4: Execute

1. On post day, check ClickUp
2. Copy content from task
3. Post to platform
4. Update status to "Posted"
5. Add metrics (impressions, engagement)

## 📝 Post Copy-Paste Guide

### LinkedIn
```
1. Open post file: post-linkedin.txt
2. Copy all content
3. Go to LinkedIn
4. Create post
5. Paste content
6. Add image
7. Post
```

### X
```
1. Open post file: post-x-thread.txt OR post-x-single.txt
2. Copy content
3. Go to X
4. Create tweet(s)
5. Paste content
6. Add image
7. Post
```

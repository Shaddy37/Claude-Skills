## ✅ Thread Checklist

- [ ] Hook is compelling (first 100 chars)

- [ ] CTA in final tweet(s)

## 📝 Notes

- Max 280 characters per tweet
- Use numbering (1, 2, 3) for scannability
- Add emojis sparingly (1-2 per tweet)
- Always include image on first and last tweet
- Link back to LinkedIn for full content

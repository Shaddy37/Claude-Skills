# 🎠 CAROUSEL_TEMPLATE.md
## LinkedIn Carousel Template

---

## Structure

### Slide 1: Title (Hook)
```
[TOPIC]
[Promise of value]

Your subtitle here
```

### Slide 2: Introduction/Problem
```
THE CHALLENGE
[What problem you're solving]

[Statistics or impact]
```

### Slide 3-X: Main Content (Steps/Points)
```
[NUMBER]. [STEP/TITLE]
[Explanation]

[Visual element]
```

### Final Slide: Summary + CTA
```
KEY TAKEAWAYS:
• Point 1
• Point 2
• Point 3

[CTA]

[Engagement question]
```

---

## Dimensions

- **Size**: 1080 x 1350 pixels
- **Ratio**: 4:5
- **Format**: PDF or image sequence

---

## Example: "5 Steps to Automate Any Process"

### Slide 1: Title
```
5 STEPS TO AUTOMATE
ANY BUSINESS PROCESS

Stop wasting 20+ hours/week
→ Save time. Save money.
```

### Slide 2: Problem
```
THE PROBLEM

Most businesses:
• Spend hours on repetitive tasks
• Pay for tools doing 80% of what they need
• Lose money every month

Sound familiar?
```

### Slide 3: Step 1
```
STEP 1: MAP

Document every step:
• Decisions
• Actions
• Outcomes

Don't skip this!
```

### Slide 4: Step 2
```
STEP 2: IDENTIFY

Find what's:
→ Repetitive
→ Rule-based
→ Time-consuming
```

### Slide 5: Step 3
```
STEP 3: DESIGN

Create workflow:
• Choose tools (n8n, Make)
• Map the flow
• Plan exceptions
```

### Slide 6: Step 4
```
STEP 4: BUILD

Start simple:
• Build one piece
• Test thoroughly
• Add complexity later
```

### Slide 7: Step 5
```
STEP 5: ITERATE

Run and improve:
• Monitor results
• Fix issues
• Optimize regularly
```

### Slide 8: Summary + CTA
```
KEY TAKEAWAYS

✓ Start small
✓ Test thoroughly
✓ Iterate always

What's YOUR biggest time drain?
Comment below 👇

Save for later!
```

## Best Practices

| Element | Guideline |
|---------|-----------|
| Title | Bold, clear, promise value |
| Content | 5-8 slides max |
| Text | 10-20 words per slide |
| Visuals | Icons, arrows, numbers |
| CTA | Question or engagement |

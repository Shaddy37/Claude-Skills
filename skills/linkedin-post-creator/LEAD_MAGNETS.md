## 📝 Lead Capture System

### Step 1: CTA in Post

```
Comment '[KEYWORD]' and I'll send you [LEAD MAGNET]
```

### Step 2: Collect Comment

```
Lead comments with keyword
[You collect: name, profile link]
```

### Step 3: DM Delivery

```
DM the lead magnet
[PDF or link]
```

### Step 4: Follow Up

```
2-3 days later:
"Hi [NAME], did you find the [MAGNET] helpful?
Do you have any questions?"
```

## 📂 Delivery Method

### Option 1: Direct DM (Easiest)

1. Person comments keyword
2. You DM the PDF/link
3. Follow up in 2-3 days

### Option 2: Email List (Better long-term)

1. Collect email via Google Form
2. Send via email sequence
3. Track opens/clicks

### Option 3: Automated (Best)

1. Set up keyword auto-response
2. Use automation to deliver
3. Track everything

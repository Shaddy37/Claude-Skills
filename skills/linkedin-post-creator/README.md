# 📝 LinkedIn Post Creator Skill

Create high-authority LinkedIn + X posts with strategic visuals, hooks, and CTAs for AI Automation Experts.

---

## 🎯 What It Does

- Creates **LinkedIn posts** with image prompts
- Creates **X/Twitter posts** (threads + single)
- Uses **strategic frameworks** for maximum engagement
- Follows **PURPLE** brand theme (#7B2CBF, #9D4EDD)

---

## 🚀 Usage

```
/linkedin [topic or intent]
```

**Examples:**
```
/linkedin agent skills
/linkedin case study
/linkedin workflow automation
/linkedin myth buster
```

---

## ⚠️ Critical Rules

1. **ALWAYS create BOTH LinkedIn AND X posts** - Never skip one
2. **2x/day posting** - Morning (Educational) + Afternoon (Engagement)
3. **Always generate image prompts** - From IMAGE_STYLES.md

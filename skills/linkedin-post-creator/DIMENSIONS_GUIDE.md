## Platform Image Dimensions & Specifications

---

## 📱 LinkedIn

| Type | Dimensions | Aspect Ratio | Best For |
|------|------------|--------------|----------|
| Single Post | 1080 x 1080 | 1:1 | Most posts |
| Single Post (Portrait) | 1080 x 1350 | 4:5 | Carousel preview |
| Carousel | 1080 x 1350 | 4:5 | All carousel slides |
| Story | 1080 x 1920 | 9:16 | Stories |
| Cover Photo | 1128 x 191 | N/A | Personal profile |
| Banner | 1584 x 396 | 4:1 | Company page |
| Article Header | 1280 x 720 | 16:9 | Blog articles |

### LinkedIn Best Practices
- Use 4:5 for maximum visibility
- Avoid text-heavy images (algorithm may reduce reach)
- Min 350 pixels per side

## 🐦 X (Twitter)

| Type | Dimensions | Aspect Ratio | Best For |
|------|------------|--------------|----------|
| Single Image | 1080 x 1080 | 1:1 | Most tweets |
| Header | 1500 x 500 | 3:1 | Profile banner |
| Card Image | 800 x 418 | 1.9:1 | Link cards |
| Preview | 1200 x 675 | 16:9 | Video thumbnails |
| Story | 1080 x 1920 | 9:16 | Fleets |

### X Best Practices
- Always use square (1:1) for images
- Header displays differently on mobile vs desktop
- Keep text center (mobile crops edges)

## 🎯 Image Generation Rules

### For Each Post

1. **LinkedIn First**: Design for 4:5 (1080x1350)
2. **X Adaptation**: Center-crop or create separate 1:1 version
3. **Text Placement**: Keep important content center (X crops edges)
4. **Minimum Size**: 600x600 for quality

### Safe Zones

**LinkedIn**:
- Most content visible
- Good margins on all sides

**X**:
- Safe zone: Center 1080x800
- Text higher (mobile shows from center)
- Avoid edges for important text

## 🔄 Converting Between Formats

### LinkedIn → X

```
4:5 (1080x1350) → 1:1 (1080x1080)
- Crop top and bottom
- Keep center content
- Resize to square
```

### X → LinkedIn

```
1:1 (1080x1080) → 4:5 (1080x1350)
- Add top/bottom padding
- Use brand color
- Or create new design
```

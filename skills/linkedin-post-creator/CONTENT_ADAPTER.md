# 🔄 CONTENT_ADAPTER.md
## LinkedIn to X Content Conversion Guide

---

## 📋 Overview

This guide helps adapt LinkedIn posts into X (Twitter) content. The goal is to:
- Save time by leveraging existing content
- Optimize for each platform's unique format
- Maintain consistent messaging across platforms

---

## 🔑 Key Principles

### 1. Don't Cross-Post Directly

| LinkedIn | X |
|----------|---|
| Long-form (1300-2000 chars) | Short (280 chars) |
| Carousel | Thread |
| Detailed case study | Tweet thread |
| Multiple images | Single image |

### 2. Adapt, Don't Duplicate

- Same message
- Different format
- Platform-specific hooks

### 3. Always Link Back

- X → LinkedIn for full content
- Drives traffic to higher-converting platform

## 🔧 Quick Conversion Rules

### LinkedIn → X

| LinkedIn Element | X Adaptation |
|-----------------|--------------|
| 1300+ chars | Thread (5-10 tweets) |
| Carousel | Thread (1 tweet per slide) |
| Long case study | Thread (7 tweets) |
| 3-4 paragraphs | 2-3 punchy tweets |
| Multiple CTAs | Single CTA at end |
| Detailed process | Step-by-step thread |

### Character Limits

| X Tweet Type | Limit |
|--------------|-------|
| Standard | 280 chars |
| With image | 280 chars |
| Thread | Unlimited |

## 📊 Output Structure

For each post, create:

```
[post-folder]/
├── post-linkedin.txt       # Full LinkedIn
├── post-x-thread.txt       # 5-10 tweet thread
├── post-x-single.txt       # 280-char single
├── image-linkedin.png      # 1080x1350
└── image-x.png            # 1080x1080
```

- [ ] Image created for X

---

## 💡 Pro Tips

1. **Write X first**: Sometimes easier to expand than contract
2. **Thread the carousel**: Each slide = one tweet
3. **Link always**: Every X thread should link to LinkedIn
4. **Images matter**: X rewards tweets with images (2x engagement)
5. **Test**: Try different hook styles, track what works

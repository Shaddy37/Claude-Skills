# 📱 Stories Templates

## Why Stories Matter

- **24-hour lifespan** = FOMO
- **Direct engagement** = Strong connection
- **Algorithm bonus** = More reach
- **No perfection needed** = Authentic

---

## Story Types

### Type 1: Behind the Scenes

**Use for:** Humanize your brand, show real work

**Content Ideas:**
- Your workspace
- Client calls
- Building something
- Office tour
- Day in the life

**Template:**
```
[IMAGE/VIDEO]

Caption:
[Brief context about what you're showing]

#BTS #BehindTheScenes
```

---

### Type 2: Quick Tips

**Use for:** Share micro-value, stay top of mind

**Content Ideas:**
- One tool tip
- Quick hack
- Productivity trick
- Little-known feature

**Template:**
```
[IMAGE WITH TIP]

Caption:
💡 Quick tip: [TIP]

Save this 💾

#Tips #QuickTips
```

---

### Type 3: Poll

**Use for:** Engagement, market research

**Content Options:**
- Binary choices
- Either/or
- This or that

**Template:**
```
[POLL STICKER]

Caption:
[Context about the question]

Vote! 👆

#Poll #Engagement
```

---

### Type 4: Q&A

**Use for:** Build authority, engagement

**Setup:**
```
Story with QUESTION STICKER:
"[YOUR QUESTION]"

Caption:
"Ask me anything!"

[ANSWER STORIES:]
[Answer with TEXT STICKER or REPLY]
```

**Questions to Ask:**
- "What's your biggest [PAIN POINT]?"
- "Tool vs manual: What do you prefer?"
- "What should I cover next?"
- "How can I help you?"

---

### Type 5: Countdown

**Use for:** Create urgency, announcements

**Use Cases:**
- New post dropping
- Live session
- Workshop/webinar
- Product launch

**Template:**
```
[COUNTDOWN STICKER]
"New [CONTENT] in [TIME]"

Caption:
"Set a reminder 🔔"

#ComingSoon #StayTuned
```

---

### Type 6: Link Sticker

**Use for:** Drive traffic, lead capture

**Best Links:**
- Latest post
- Free resource
- Blog/video
- Booking link

**Template:**
```
[IMAGE + LINK STICKER]

Caption:
"👆 Tap for [VALUE]"

[What they'll get]

#LinkInBio #FreeResource
```

---

### Type 7: Reaction/Response

**Use for:** Engage with audience, community

**Template:**
```
[SCREENSHOT OF COMMENT/DM]

[YOUR REPLY via REPLY STICKER or TEXT]

"Great question! [ANSWER]"

#Community #Engagement
```

---

---

### Type 9: Slider

**Use for:** Fun engagement, opinions

**Template:**

## Story Best Practices

### Posting Frequency
- Post **3-5 stories daily**
- **Morning** (7-9 AM): Tips, motivation
- **Midday** (12-2 PM): Engagement, Q&A
- **Evening** (6-9 PM): BTS, personal

### Engagement Tips
- Always include **CTAs**
- Reply to **all** story replies
- Use **all** interactive stickers
- Add **music** when relevant

### Design Tips
- Keep text **big and readable**
- Use **high contrast**
- Add **background color**
- Stay **on brand** (colors, fonts)

### Story Features to Use
| Feature | Purpose |
|---------|---------|
| Poll | Engagement |
| Q&A | Authority + engagement |
| Quiz | Education + fun |
| Countdown | Urgency |
| Link | Traffic/lead gen |
| Reaction | Community |
| Slider | Fun engagement |

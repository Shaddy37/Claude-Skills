# 📋 Post Types - Instagram Formats

## 1. Carousel (Highest Engagement)

### When to Use
- Educational content
- Tutorials
- Listicles
- Step-by-step guides
- Process breakdowns

### Structure
```
Slide 1: HOOK - "5 Ways to Automate [Topic]"
Slide 2: Way 1 - Brief intro
Slide 3: Way 2 - With visual
Slide 4: Way 3 - With example
Slide 5: Way 4 - Key tip
Slide 6: Way 5 + RESULT + CTA
```

### Carousel Caption Template
```
[HOOK - First 2 lines]

Swipe to see exactly how to [RESULT] 👆

[Context - Brief background]

[Key points from carousel]

What's your biggest challenge with [TOPIC]? 👇

Save this post for later 📌

#hashtags
```

### Carousel Best Practices
- 5-10 slides optimal
- Each slide = one idea
- Use numbered progression
- Consistent design
- End with result + CTA

---

## 2. Reels (Highest Reach)

### When to Use
- Quick tutorials
- Tool demos
- "POV" content
- Before/after
- Tips in 30 seconds

### Structure
```
0-3 sec: HOOK (stop scroll)
3-15 sec: VALUE (main content)
15-30 sec: CTA (follow + engagement)
```

### Reels Script Template
```
[HOOK]
"POV: You just discovered [TOOL] and it changed everything..."

[CONTENT]
"Here's how [RESULT] in just [TIME]:"

[STEP 1]
First, [ACTION]...

[STEP 2]
Then, [ACTION]...

[CTA]
"Follow for more automation tips 💡"
```

### Reels Cover Image
- Bold headline text
- High contrast colors
- 1080x1920 (9:16)
- Stop-scroll energy

---

## 3. Single Image Post

### When to Use
- Quotes
- Results/metrics
- Tool stack
- Before/after
- Quick tips
- Announcements

### Types

#### Quote Post
```
Image: Your quote + author name
Caption: Context + question + hashtags
```

#### Result Post
```
Image: Big metric + context
Caption: Brief story + CTA
```

#### Tool Stack
```
Image: Logos in grid
Caption: Why these tools + CTA
```

### Single Post Caption Template
```
[HOOK - Line 1]

[BODY - Main message]

[QUESTION - Engagement]

[CTA]

#hashtags
```

---

## 4. Stories (High Engagement)

### When to Use
- Behind the scenes
- Quick tips
- Polls/Q&A
- Countdowns
- Updates

### Story Types

#### Poll Story
```
Image: Question + 2 options
Caption: Results + follow-up
```

#### Q&A Story
```
Image: Question box
Answer: Text overlay or reply
```

#### BTS Story
```
Image/Video: Real work moment
Caption: Context
```

#### Quick Tip Story
```
Image: Single tip + visual
Caption: "Save this 💡"
```

### Story Best Practices
- Post 3-5 stories daily
- Use all interactive features
- Add music when relevant
- Include CTA (link, question)

---

## 5. Text Post (New IG Feature)

### When to Use
- Opinions
- Hot takes
- Short thoughts
- Reflections

### Text Post Template
```
[HOOK - First line]

[BODY - Main thought]

[QUESTION]

#hashtags
```

### Text Post Tips
- Keep under 500 words
- Use line breaks
- Bold key points
- End with question

## Post Selection Guide

| Goal | Best Format |
|------|-------------|
| **Saves** | Carousel |
| **Reach** | Reels |
| **Comments** | Carousel + Questions |
| **Follows** | Reels + Value |
| **DMs** | Carousel + CTA |

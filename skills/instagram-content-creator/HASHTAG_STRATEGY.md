## Hashtag Placement

### In Caption
```
First 3 lines: Most important (niche)

Body: Context

Last lines: Save CTA + hashtags
```

### Example
```
5 Ways to Automate Your Business 👇

[CAROUSEL CONTENT]

Save this for later 📌

#AIautomation #n8n #workflowautomation #aibusiness #automationtips #productivityhacks #futureofwork #businessautomation #learnontiktok #education
```

### After First Comment
```
Caption: No hashtags

Comment 1: #hashtag1 #hashtag2 #hashtag3...
```

## Hashtag Mistakes to Avoid

### ❌ Don't Do
1. Don't use banned hashtags
2. Don't use same hashtags every post
3. Don't overstuff (30+ looks spammy)
4. Don't use unrelated hashtags
5. Don't use only huge hashtags
6. Don't forget to rotate

### ✅ Do Instead
1. Mix small + medium + large
2. Change 30-50% each post
3. Use niche-specific tags
4. Research new hashtags weekly
5. Track which hashtags work

---

## Finding New Hashtags

### Methods
1. **Competitors** - What are they using?
2. **Related posts** - Check explore page
3. **Hashtag tools** - Display purposes, Hashtag Guru
4. **Instagram suggestions** - After posting
5. **Trend alerts** - What's rising?

### Hashtag Research Checklist
- [ ] Relevance to content
- [ ] Post count (not too high, not too low)
- [ ] Competition level
- [ ] Recent posts quality
- [ ] Potential reach

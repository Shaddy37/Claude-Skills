# 🎨 Image Styles Guide

## Image Sizes (2026)

| Format | Dimensions | Aspect Ratio | Use |
|--------|------------|--------------|-----|
| **Feed Square** | 1080x1080 | 1:1 | Single posts, quotes |
| **Feed Portrait** | 1080x1350 | 4:5 | Carousels, single posts |
| **Stories/Reels** | 1080x1920 | 9:16 | Vertical video covers |
| **Landscape** | 1080x608 | 1.91:1 | Links ads (rare) |

## Image Types & Prompts

### Type 1: Carousel Slide

**Use:** Educational content, step-by-step guides

**Dimensions:** 1080x1350

**Prompt Template:**
```
"[MAIN TOPIC], slide format, dark theme (#1a1a2e) with cyan accent (#00d4ff), numbered circle [NUMBER], modern tech aesthetic, clean minimal design, professional business style, Instagram carousel, 1080x1350px"
```

**Example:**
```
"5 Ways to Automate Your Business, numbered circles 1-5, dark theme (#1a1a2e) with cyan accents (#00d4ff), modern tech aesthetic, minimalist professional, Instagram carousel format, 1080x1350px"
```

## Design Elements

### Text Styles
- **Headlines:** Bold, 48-72pt
- **Subtext:** Regular, 24-36pt
- **Numbers:** Extra Bold, 96-144pt

### Visual Elements
- **Icons:** Simple, line style
- **Arrows:** Gradient cyan
- **Circles:** Rounded, accent colors
- **Backgrounds:** Solid or subtle gradient

---

## Nano Banana Generation Tips

1. **Start with subject** - What the image is about
2. **Add style** - Dark theme, modern, tech
3. **Specify colors** - Hex codes work best
4. **Include format** - Instagram post, carousel, Reels cover
5. **Add dimensions** - 1080x1350, 1080x1920, etc.
6. **Keep it simple** - Too many elements = cluttered

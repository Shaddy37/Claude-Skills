# 📸 Instagram Content Creator Skill

Create high-engagement Instagram posts, carousels, Reels, and Stories for AI Automation Experts.

---

## 🎯 What It Does

- Creates **Carousels** (educational, high saves)
- Creates **Reels** (reach, discovery)
- Creates **Single Posts** (quick value)
- Creates **Stories** (engagement)
- Follows **PURPLE** brand theme (#7B2CBF, #9D4EDD)

---

## 🚀 Usage

```
/instagram [topic or intent]
```

**Examples:**
```
/instagram agent skills
/instagram carousel
/instagram reel
/instagram automation tips
```

---

## ⚠️ Critical Rules

1. **ALWAYS ask strategic questions first** - Topic, Format, Goal, Image Style
2. **Use PURPLE theme** - NOT green/cyan
3. **Generate image prompts** - From IMAGE_STYLES.md

## 📝 Content Formats

| Format | Best For | Template |
|--------|----------|----------|
| Carousel | Saves, Educational | 7-10 slides |
| Reels | Reach, Discovery | 30-60 seconds |
| Single Post | Quick Value | Image + Caption |
| Stories | Engagement | 24hr expiry |

---

## 🎨 Image Style

**Theme:** PURPLE (#7B2CBF, #9D4EDD)
- Clean, minimalist aesthetic
- Bold typography
- Professional, not generic

---

## 📅 Content Strategy

- **Educational content** → Carousels (high saves)
- **Discovery content** → Reels (high reach)
- **Engagement** → Stories, Questions
- **Authority** → Client results, Proof

---

## 📦 Output Files (per post)

- `post-instagram.txt` - Instagram caption
- `carousel-slides/` - Slide content
- `image-prompt.txt` - Image generation prompt
- `hashtags.txt` - Hashtag bundle
- `reel-script.txt` - Reel script (if Reels)

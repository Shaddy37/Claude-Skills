## 🔄 Generation Workflow

### Phase 1: Strategic Inquiry (Ask These Questions)
1. **Topic**: What is the post about?
2. **Format**: Carousel, Reel, Single, or Story?
3. **Goal**: Saves (educational), Reach (Reels), or Engagement (comments)?
4. **Image Style**: Infographic or Personal photo with overlay?

### Phase 2: Library Selection
1. **Hooks**: Pull from [HOOKS_COLLECTION.md](file:///e:/Antigravity/Shadab/.claude/skills/instagram-content-creator/HOOKS_COLLECTION.md)
2. **Templates**: Use appropriate template based on format
3. **Hashtags**: Pull from [HASHTAG_STRATEGY.md](file:///e:/Antigravity/Shadab/.claude/skills/instagram-content-creator/HASHTAG_STRATEGY.md)
4. **CTAs**: Select from [CTA_LIBRARY.md](file:///e:/Antigravity/Shadab/.claude/skills/instagram-content-creator/CTA_LIBRARY.md)

### Phase 3: Selective Build
- **NOT every post needs everything**
- Carousel → Educational content + save CTA
- Reels → Hook + quick value + follow CTA
- Single → Result/metric + engagement CTA
- Stories → Interactive + response CTA
